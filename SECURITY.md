# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.0   | :white_check_mark: |

## Reporting a Vulnerability

Please [report vulnerabilities](https://github.com/coignard/u880/security/advisories/new) rather than opening a public issue. Thanks!
